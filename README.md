# Generative-AI-Tools

* テキスト生成AI
    * [ChatGPT](https://chat.openai.com/)
    * [Gooble Bard](https://bard.google.com/chat)
    * [Claude2](https://claude.ai/chats)（クロード2）
    
* 音楽生成AI
    * [CREEVO](https://creevo-music.com/)（クリーボ）  
        1. テキスト生成AIを使って作詞（12小節まで）
        1. Googleアカウントでログイン
        1. [デザイン作曲] を選択
        1. [コード進行] を設定
            * 楽曲の全体的な特徴
            * 調
        1. [メロディのスタイル] を設定
            * メロディの雰囲気
            * テンポBPM
            * メロディ楽曲
            * メロディ生成の設定
                * 音域
                * 遊び度
                * 和音の当てはまり度
            * 伴奏のスタイル（伴奏なしも選択可）
                * 伴奏パターン1,2
                * 伴奏楽器
                * ドラムパターン
        1. インストゥルメンタル版で保存（.mp3）
